# StepNest
A decentralized walking and hiking tracker with social features and route suggestions built on Stacks blockchain.

## Features
- Track and store walking/hiking routes
- Share routes with other users
- Rate and review routes
- Earn tokens for completing routes
- Create and suggest new routes

## Smart Contracts
- StepNestCore: Main contract handling routes and user interactions
- StepToken: Reward token for route completion
